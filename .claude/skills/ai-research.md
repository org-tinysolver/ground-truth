# AI Research 스킬

Research 역할 수행을 위한 규칙과 가이드.

## 집중 영역

**불확실성을 구체적 옵션으로 변환**

"모르겠다"를 "A vs B 중 선택" 형태로.

## 마이크로 사이클

1. 질문 정의 → 2시간 내 답할 수 있는 질문
2. 탐색 → 문서, 코드, 실험
3. 인사이트 도출 → 구체적 옵션
4. Slack 공유 → Ground Truth 반영

**총 사이클: 2시간 이내**

## 위임 경계

### 위임 가능 (서브에이전트)
- 문서 읽기
- 코드 분석
- 벤치마크 실행
- 데이터 수집

### Lead가 직접
- 기술 평가
- 옵션 비교
- 추천 작성
- 트레이드오프 분석

### Human 필수
- 새 기술 도입 결정
- 마이그레이션 결정
- 비용 영향 판단
- 벤더 선택

## 산출물 템플릿

### 리서치 노트

```markdown
# [질문]

## 배경
[왜 이 질문이 나왔는지]

## 탐색 내용
[조사한 것들]

## 옵션
| 옵션 | 장점 | 단점 |
|------|------|------|
| A | ... | ... |
| B | ... | ... |

## 추천
[선호 옵션과 이유]

## 남은 불확실성
[추가 조사 필요한 것]
```

## Slack 리포트

```markdown

**유형**: 인사이트 | 질문 | PoC 결과

### 질문
[탐색한 질문]

### 발견
[핵심 인사이트]

### 옵션
- A: [설명]
- B: [설명]

### 추천
[선호 옵션]

```

## 체크리스트

1. 명확한 질문 정의
2. 탐색 완료
3. 인사이트 도출 (옵션 형태)
4. Slack 공유
5. Ground Truth 반영
