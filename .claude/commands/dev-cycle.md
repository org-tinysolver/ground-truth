# Dev 사이클 시작

마이크로 Dev 사이클을 시작합니다.

## 사용법

```
/dev-cycle <스펙 또는 이슈>
```

## 실행 순서

ai-dev 스킬을 활성화하고 다음을 수행:

1. **스펙 확인** - 무엇을 만들어야 하는지 명확히
2. **최소 구현** - 핵심만 먼저
3. **테스트** - 즉시 실행
4. **Slack 리포트** - 결과 공유

## Human 체크

시작 전 확인 - 하나라도 Yes면 멈추고 Human 승인 요청:
- [ ] 되돌릴 수 없는 작업?
- [ ] 프로덕션 영향?
- [ ] 데이터 삭제?
- [ ] 보안 관련?
- [ ] 비용 발생?
