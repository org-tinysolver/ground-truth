# 위임 레벨 확인

현재 작업의 위임 레벨을 확인합니다.

## 사용법

```
/check-delegation <작업 설명>
```

## 판단 기준

### Level 1: 서브에이전트 (진행)
- 반복적 작업
- 명확한 규칙
- 되돌릴 수 있음

### Level 2: Lead (진행, 신중히)
- 판단 필요
- 트레이드오프
- 팀 영향

### Level 3: Human (멈추고 승인 요청)
- 되돌릴 수 없음
- 비용 발생
- 외부 영향
- 보안 관련
- 사용자 데이터

## 체크리스트

```
□ 되돌릴 수 없는가? → Human
□ 비용이 발생하는가? → Human
□ 외부에 영향을 주는가? → Human
□ 보안 관련인가? → Human
□ 사용자 데이터를 다루는가? → Human
```

**의심되면 → Human에게 확인**
