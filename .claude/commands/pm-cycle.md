# PM 사이클 시작

마이크로 PM 사이클을 시작합니다.

## 사용법

```
/pm-cycle <문제 또는 기능>
```

## 실행 순서

ai-pm 스킬을 활성화하고 다음을 수행:

1. **문제 정의** - 1문장으로 정리
2. **가설 수립** - 검증 가능한 형태
3. **마이크로 PRD 작성** - 템플릿 사용
4. **Slack 리포트 준비** - 승인 요청 형태

## 체크

시작 전 확인:
- [ ] Ground Truth 컨텍스트 확인했는가?
- [ ] 2시간 이내 검증 가능한 범위인가?
- [ ] Human 승인 필요 여부 판단했는가?
