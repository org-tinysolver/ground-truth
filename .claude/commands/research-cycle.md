# Research 사이클 시작

마이크로 Research 사이클을 시작합니다.

## 사용법

```
/research-cycle <질문>
```

## 실행 순서

ai-research 스킬을 활성화하고 다음을 수행:

1. **질문 정의** - 2시간 내 답할 수 있는 질문으로
2. **탐색** - 문서, 코드, 실험
3. **옵션 도출** - "A vs B" 형태로
4. **Slack 공유** - 인사이트 리포트

## 산출물

리서치 노트 형식으로 정리:
- 배경
- 탐색 내용
- 옵션 비교 테이블
- 추천
- 남은 불확실성
