# POC 가이드

Proof of Concept - 아이디어를 빠르게 검증하는 방법

## POC란?

> "완벽한 실패보다 빠른 실패가 낫다"

POC는 아이디어의 **기술적 가능성**과 **비즈니스 가치**를 빠르게 검증하는 단계입니다.

## POC 원칙

```mermaid
mindmap
  root((POC 원칙))
    속도
      1-2주 내 완료
      완벽보다 빠름
    집중
      핵심 가설 1개
      불필요한 것 제거
    검증
      될까? 안될까?
      Go/No-Go 결정
    버리기
      아깝지 않게
      다시 시작할 용기
```
