# 오픈 기여 가이드

내부 오픈소스 스타일로 제품을 확장하는 방법

## 오픈 기여란?

만들어진 제품을 팀 전체에 공개하고, 누구나 기여할 수 있도록 하는 문화입니다.

```mermaid
flowchart LR
    A[1인 개발] --> B[팀 공개]
    B --> C[기여 환영]
    C --> D[함께 성장]
```
