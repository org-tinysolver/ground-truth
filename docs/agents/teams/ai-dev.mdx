# AI Dev

구현, 테스트, 배포를 담당하는 AI 팀.

## 역할

- 스펙 기반 구현
- 테스트 작성 & 실행
- 배포 & 모니터링

## 마이크로 사이클

```mermaid
flowchart LR
    A[스펙 수신] --> B[최소 구현]
    B --> C[테스트]
    C --> D{통과?}
    D -->|Yes| E[Slack 리포트]
    D -->|No| F[수정]
    F --> B
    E --> G[다음 사이클]
```

### 사이클 원칙

| 원칙 | 설명 |
|------|------|
| 최소 구현 | 스펙의 핵심만 먼저 구현 |
| 즉시 테스트 | 구현 즉시 테스트 가능해야 함 |
| 빠른 피드백 | 2시간 이내 검증 가능한 단위 |

## 위임 구조

```mermaid
flowchart TB
    DEV_LEAD[Dev Lead]
    FE[Frontend Agent]
    BE[Backend Agent]
    TEST[Test Agent]

    DEV_LEAD --> FE
    DEV_LEAD --> BE
    DEV_LEAD --> TEST
```

```markdown
- ❌ 실패: [N개]

### 다음 단계
[다음에 할 작업]
```
