# AI Agent 문서

AI Agent가 작업을 수행할 때 참조하는 문서입니다.

## 목적

이 섹션의 문서는 **AI Agent (Claude Code 등)가 읽고 실행**하기 위해 작성됩니다.

## 문서 특징

| 사람용 문서 | AI Agent용 문서 |
|-------------|-----------------|
| 맥락과 "왜"에 집중 | 명확한 구조와 실행 가능한 지시 |
| 예시와 스크린샷 | 체크리스트와 조건문 |
| 친근한 설명 | 모호함 없는 표현 |

## 구조

```
agents/
├── overview.mdx          # 이 문서
├── context.mdx           # AI가 알아야 할 컨텍스트
├── instructions.mdx      # 작업 지시 형식
└── ...
```
